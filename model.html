Model

順位予測モデルの前提条件と採用パラメータ

モデル概要

本サイトでは、Jリーグの試合結果やシュート数をもとに各チームの攻撃力・守備力を推定し、 ポアソン分布を用いて各試合の得点確率を計算しています。 その結果をもとにリーグ戦を複数回シミュレーションし、最終順位を予測します。

使用データ

  • 対象リーグ：J1リーグ
  • 対象シーズン：2024年・2025年
  • 使用データ：得点、失点、シュート数、枠内シュート数
  • 評価指標：MAE（平均順位誤差）

前提条件・採用パラメータ

項目 設定 理由
試合モデル ポアソン分布 サッカーの得点数のような低頻度イベントを扱いやすいため
更新間隔 3試合ごと 1試合ごとよりもノイズを抑え、5試合ごとよりも変化に追従しやすかったため
期待得点上限 3.5 極端な大量得点予測を抑えるため
Decay 1.00 直近試合を過度に重視すると精度が悪化したため
ホーム補正 0.0 検証上、補正なしの方がMAEが小さかったため
フォーム補正 0.0 直近成績を混合すると精度が悪化したため
シミュレーション回数 100回 検証速度を優先し、開発段階では軽量に運用するため
Elo補正 INITIAL_ELO = 1500 K_FACTOR = 16 HOME_ADV = 0 ELO_LAMBDA_WEIGHT = 0.20 この組み合わせであくまでλへの弱い補正として用いる方がMAEに改善が見られたため
相性Effect 相性Effect0.20 時間減衰0.97 縮小補正30 Eloレーティングの使用方法のの改良が見込めるため

検証結果の例

これまでの検証では、更新間隔は3試合ごと、期待得点上限は3.5、 Decayは1.00が比較的良い結果となりました。 今後はシュート数や枠内シュート数、前年レーティングを組み合わせ、 予測精度の改善を進めていきます。
