順位予測モデルの検証・改善記録

開発ログについて

本ページでは、Jリーグ順位予測モデルの開発過程で行った検証内容を記録しています。 予測結果だけでなく、どの仮説を試し、どの設定を採用・不採用にしたのかを残すことで、 モデル改善の過程を可視化することを目的としています。

2026/05/18　v0.0

サッカー順位予測モデルの構想

内容

サッカーの勝敗や順位をAI・統計モデルで予測する方法について検討を開始。 まずは試合結果データを用いて、チームの強さを数値化し、リーグ戦をシミュレーションする方向性を定めた。

位置づけ

現在のFootball Prediction Labの原型となる構想段階。

2026/05/20　v0.1

基本モデルの検証

仮説

試合結果から各チームの攻撃力・守備力を推定し、ポアソン分布を用いることで、 リーグ後半戦の順位をある程度予測できるのではないか。

検証内容

  • J1の試合結果CSVを使用
  • 前半戦の成績からチームの攻撃力・守備力を推定
  • 後半戦を複数回シミュレーション
  • 実際の最終順位との誤差をMAEで評価

結果

20試合終了時点の検証で、MAEはおおむね2.5前後となった。 単純なモデルでも一定の予測力があることを確認した。

2026/05/19　v0.2

更新間隔の比較

仮説

毎試合ごとにレーティングを更新すると、直近の偶然性に影響されすぎる可能性がある。 一方で、更新間隔が長すぎるとチーム状態の変化に追従できない可能性がある。

比較条件

更新間隔 評価
1試合ごと 直近結果に敏感すぎる可能性
3試合ごと ノイズと追従性のバランスが良い
5試合ごと 変化への反応が遅くなる可能性

結論

検証の結果、3試合ごとの更新が比較的良い結果となったため、 現行モデルではUPDATE_EVERY = 3を採用している。

2026/05/19　v0.3

Decayの検証

仮説

直近の試合ほどチームの現在の強さを反映しているため、 勢いによる連勝・引きずる連敗を再現できる可能性 過去の試合に小さい重みを与えることで予測精度が向上する可能性がある。

比較条件

Decay 結果の傾向
0.95〜0.99 直近試合を重視
1.00 全試合を同じ重みで扱う

結論

直近試合を重視する設定では精度が改善せず、Decay = 1.00が最も安定した。 そのため、現行モデルでは過去試合に減衰重みをかけていない。

2026/05/25　v0.4

期待得点上限の検証

仮説

ポアソン分布では、チーム力の差が大きい場合に期待得点λが過大になり、 現実的でない大量得点の確率が高くなる可能性がある。 いくら実力差があっても5点以上をとることは意外と少なく、3-0程度で無理に攻めず終わることの方が多い。これを再現できる可能性。

比較条件

期待得点上限 結果の傾向
なし 極端なスコアが出やすい
2.5〜3.0 やや抑制が強い
3.5 最も良好
4.0 上限としてはやや緩い

結論

期待得点λ上限はLAMBDA_CAP = 3.5を採用した。

仮説

ホームチームには一定の優位性があるため、期待得点λに補正を加えることで予測精度が向上する可能性がある。

結果

検証上、ホーム補正を加えない設定の方がMAEが小さかった。

結論

現行モデルではホーム補正を0.0としている。

2026/06/01　v0.6

フォーム補正の検証

仮説

直近数試合の好不調をモデルに混合することで、現在のチーム状態をより反映できる可能性がある。 Decayとは違い、ここ数試合のみ結果をλ係数に加える方法で、大きな波でなく2~3試合の連勝・連敗の再現を図りたい。

結果

フォーム補正を加えると、むしろ予測精度が悪化した。 直近の勝敗には偶然性が大きく含まれるため、過度に反映するとノイズになる可能性がある。

結論

現行モデルではフォーム補正を0.0としている。

2026/06/18　v1.2

ver.1.2の結果からみる今後の方針

考察と方針

  • 柏・町田の過小評価
    両チームは接戦を勝ち切る試合が多く、モデル上では実際の勝点ほど高く評価されていない。 特に柏は、前半時点の内容指標から「上振れ」と判定され、後半戦で失速すると見積もられた可能性がある。 今後は、枠内シュート差・失点の少なさ・接戦での勝点獲得能力をどのように扱うかを検討する。
  • 川崎Fの過大評価
    川崎Fは大量得点での勝利があり、得点力・得失点差・Eloが高く出た一方で、 接戦を落とす試合も多かった可能性がある。 ポアソンモデルが「大量得点による強さ」をやや過大に評価している可能性を検証する。
  • 昇格クラブの扱い
    前年J1データが存在しない昇格クラブについて、前年由来の攻守係数を中立値1.0として混ぜていた。 この処理が清水・岡山・横浜FCの評価を歪めている可能性があるため、 昇格クラブは前年重みを0にする案を検証する。
  • FC東京の過小評価
    FC東京はシュート数や得点結果では低く評価される一方、xGでは極端に悪くない可能性がある。 現モデルでは2025年側にxGを入れていないため、xGを説明変数として追加する案を検討する。
  • 湘南・横浜FMの大きな変動
    横浜FMは後半戦で大きく改善し、湘南は後半戦で大きく失速した。 現モデルは前半戦から推定した強さを固定して後半戦を予測しているため、 シーズン中の状態変化を十分に反映できていない可能性がある。
  • Decay・ローリングモデルの再検討
    直近試合をより重視するDecay設定や、一定試合ごとに攻守係数・Eloを更新するローリングモデルを再検討する。 まずは昇格クラブの前年重み0、Decay値の比較から順に検証する。

2026/06/18 v1.3

ver.1.3：昇格クラブの前年重み補正を導入

更新内容

  • 前年J1データが存在しない昇格クラブについて、前年由来の攻守係数を中立値1.0として混ぜる処理を見直した。
  • これまでのモデルでは、昇格クラブにも前年重み0.4が適用されていたため、 実質的に「2025年前半の強さ60% + 中立値1.0を40%」として評価されていた。
  • ver.1.3では、前年J1での試合数が0のチームについて、前年重みを0.0に変更した。

検証結果

  • 元モデルではMAEが 2.3237 だったのに対し、 昇格クラブの前年重みを0にしたモデルではMAEが 2.2577 となり、予測精度が改善した。
  • 特に岡山と横浜FCの平均予測順位が実順位に近づき、昇格クラブの扱いとしては一定の改善が見られた。
  • 一方で、清水はさらに過大評価される結果となった。 そのため、清水については前年重みではなく、2025年前半の攻守係数そのものが高く出すぎている可能性がある。

考察

  • 昇格クラブに前年J1の中立値を混ぜる処理は、チームによっては評価を不自然に平準化していた可能性がある。
  • 横浜FCのように実順位が下位だったチームでは、前年重み0によって予測順位がより現実に近づいた。
  • ただし、清水のように前半戦の内容指標が高く出ているチームでは、前年重みを外すことで逆に評価が上がる場合もある。

今後の方針

  • 昇格クラブの前年重みを、0.0 / 0.1 / 0.2 / 0.3 / 0.4 の範囲で比較し、最適な補正値を検証する。
  • 清水の過大評価については、前半戦の攻守係数・枠内シュート・xGの扱いを詳しく確認する。
  • 柏・町田の過小評価、川崎Fの過大評価については、接戦の勝敗や大量得点試合の影響を検証する。
  • 直近フォームを反映するDecay設定や、一定試合ごとに強さを更新するローリングモデルも引き続き検討する。
2026/06/18 v1.4

ver.1.4：大勝補正 cap4 の導入

更新内容

  • 攻守係数を計算する際、5得点以上の大勝試合をそのまま評価に反映すると、 1試合の大量得点によって攻撃力・守備力が過度に動く可能性がある。
  • ver.1.4では、攻守係数計算用の得点に上限を設け、 5得点以上の試合を最大4得点として扱う cap4 を導入した。
  • 実際の順位表やシミュレーション上の得点は変更せず、 チームの強さを推定するための攻守係数計算にのみ大勝補正を適用した。
  • ver.1.3で導入した「昇格クラブの前年重み0」補正は引き続き採用している。

検証設定

  • GOAL_ADJUST_MODE = cap4
  • GOAL_CAP_FOR_STRENGTH = 4
  • USE_RAW_LEAGUE_AVG_FOR_LAMBDA = True
  • リーグ平均得点は生の得点を使い、チーム攻守係数のみ補正得点で計算した。

検証結果

  • ver.1.3のMAEは 2.2577 だった。
  • 大勝補正cap4を導入したver.1.4では、MAEが 2.2225 となり、予測精度が改善した。
  • 参考として、より強い補正であるcap3も検証したが、MAEは 2.2501 となり、cap4ほどの改善は見られなかった。

考察

  • cap4は、大勝試合の影響を抑えつつ、強豪チームの評価を過度に削りすぎないバランスの良い補正となった。
  • cap3では補正がやや強く、清水・名古屋・東京Vなどの過大評価が強まる傾向が見られた。
  • 一方で、川崎Fの過大評価はcap4では大きく改善しなかった。 そのため、川崎Fの評価が高く出る原因は大量得点だけではなく、 枠内シュート・Elo・前半戦の得失点差などにもあると考えられる。

今後の方針

  • ver.1.4では大勝補正cap4を採用し、攻守係数の安定化を図る。
  • 柏・町田の過小評価については、接戦での勝点獲得能力や守備安定性の扱いを検討する。
  • FC東京の過小評価、清水・名古屋の過大評価については、xGの導入や重み付けの変更を検証する。
  • 川崎Fの過大評価については、大量得点以外の攻撃指標やElo補正の影響を確認する。
2026/06/18 v1.5

ver.1.5：3年検証による基準モデルの再設定

更新内容

  • ver.1.4までは主に2025年シーズンを対象としてモデルの改良を行ってきたが、 単年度の結果だけを基準にすると、その年特有の偶然やリーグ展開に最適化されすぎる可能性がある。
  • ver.1.5では、より汎用的な順位予測モデルを目指し、 2023年・2024年・2025年の3年分を対象として係数を再検証した。
  • 具体的には、2022年の結果を用いて2023年を予測、 2023年の結果を用いて2024年を予測、 2024年の結果を用いて2025年を予測する形で検証を行った。
  • 終盤補正や監督解任ブーストなどの追加要素は入れず、 今後の比較対象となる基準モデルとして整理した。

検証設定

  • PREV_WEIGHT = 0.20
  • PREV_DECAY = 0.995
  • ELO_LAMBDA_WEIGHT = 0.20
  • COMPAT_WEIGHT = 0.20
  • DRAW_FACTOR = 1.20
  • MAX_MATCH_DRAW_PROB = 0.33
  • LAMBDA_CAP = 3.5
  • GOAL_ADJUST_MODE = cap4
  • GOAL_CAP_FOR_STRENGTH = 4

検証結果

  • 2023年・2024年・2025年の3年分を対象に、各年度の前半戦を学習データ、 後半戦を予測対象として順位予測を行った。
  • これにより、2025年単年に合わせた設定ではなく、 複数年度で安定して機能するかを確認できるようになった。
  • ver.1.5の予測結果は、別ページに3年分の順位表と年度別サマリーとして掲載する。

考察

  • 単年度のMAEだけで設定を選ぶと、特定シーズンの上振れ・下振れに影響されやすい。 そのため、複数年度で検証することで、モデルの汎用性を確認しやすくなった。
  • ver.1.5では、終盤補正や監督解任ブーストを入れない状態を基準としたため、 今後の追加補正が本当に改善につながるかを比較しやすくなった。
  • また、試合単位の予測ログも出力し、各試合の期待得点、勝敗確率、期待勝点、 実勝点との差分を確認できるようにした。

今後の方針

  • ver.1.5を基準モデルとして、監督解任ブーストの有無を検証する。
  • まずは監督解任後の試合について、ver.1.5の期待勝点に対する実勝点の上振れ・下振れを確認する。
  • そのうえで、効果が見られる場合は、攻撃λブースト・守備λ抑制・試合数減衰などの候補を比較する。
  • 監督解任ブーストを導入する場合も、3年平均MAEだけでなく、解任後試合のみの予測誤差も評価する。
2026/06/20 v1.5追加検証

ver.1.5追加検証：監督交代ブーストの検証

検証の目的

ver.1.5を基準モデルとして、シーズン途中の監督交代後にチーム成績が一時的に上振れするかを検証した。 ここで扱う日付は厳密な解任発表日ではなく、Soccer D.B.の試合記録から推定した 「新監督が確認できる最初の試合日」である。 そのため、本検証では「解任ブースト」ではなく、より広く「監督交代ブースト」として扱う。

検証方法

  • まず、2023〜2025年のJ1監督交代イベントについて、ver.1.5の試合単位ログと結合し、 監督交代後の実勝点がモデル期待勝点を上回るかを確認した。
  • 次に、前任監督の最終戦をcutoffとして、その時点までの試合結果のみでver.1.5型の期待勝点・期待得点・期待失点を再計算し、 新監督初戦以降の実績と比較する反実仮想分析を行った。
  • さらに、対象をJ1 2005〜2025年まで拡張し、118件の監督交代イベントを対象に同じcutoff反実仮想分析を行った。
  • 評価指標は、実勝点からモデル期待勝点を引いた勝点残差、実得点から期待得点λを引いた得点残差、 期待失点λから実失点を引いた失点改善を用いた。

主な検証結果

検証 対象 after_1_3 勝点残差 after_1_5 勝点残差 解釈
ver.1.5ログによる順解析 J1 2023〜2025 -0.056 -0.008 短期的な上振れは確認できなかった
cutoff反実仮想 J1 2023〜2025・10件 -0.148 -0.085 交代直後のブーストは確認できなかった
cutoff反実仮想 J1 2005〜2025・118件 +0.077 +0.086 ごく小さい上振れ傾向はあるが、信頼区間は0をまたぐ

考察

  • 2023〜2025年の直近3年だけを見ると、監督交代後1〜3試合の勝点残差はマイナスであり、 一律の短期ブーストを入れる根拠は得られなかった。
  • 一方で、2005〜2025年まで対象を広げると、after_1_3で平均+0.077、after_1_5で平均+0.086の小さな上振れが見られた。 ただし、95%信頼区間は0をまたいでおり、全クラブ共通の補正として扱うには弱い。
  • イベント単位では、2015年甲府、2018年浦和、2022年神戸、2015年鹿島など、 交代直後に大きく上振れした事例も存在した。一方で、大きく下振れした事例もあり、効果は二極化している。
  • 交代時期で見ると、シーズン中盤、特に17〜24試合消化時点の交代で上振れが出やすい可能性がある。 ただし、これは条件探索段階の仮説であり、正式な補正として採用するには追加検証が必要である。
  • 後任監督が内部昇格・暫定監督・クラブ事情を把握しているタイプかどうかも、効果の有無に関係している可能性がある。 ここは自動判定が難しいため、今後は手動ラベルを追加して検証する。

結論

  • 現時点では、ver.1.5に一律の監督交代ブーストは採用しない。
  • ただし、2005〜2025年の長期データでは小さな上振れ傾向が見られたため、 監督交代を完全に無視するのではなく、条件付き補正の候補として継続検証する。
  • 次の検証では、交代時期、チーム状態、相手日程、後任監督タイプを条件として、 どの条件であればブースト補正を入れる価値があるかを検討する。
